# 🌈 Holi Wish Generator

Create and share **personalized Holi wishes** with a unique link.

This simple web app allows users to write a Holi message, generate a shareable link, and send it to friends and family.

🔗 Live Demo  
https://holii-tawny.vercel.app/wish

---

## ✨ Features

- 🎨 Add your own Holi message
- 🔗 Generate a unique wish link
- 📋 Copy the link instantly
- 📱 Mobile-friendly design
- ⚡ Fast and simple UI

---

## 🚀 How It Works

1. Enter your **Holi message**
2. Click **Generate Link**
3. Copy the generated link
4. Share it with your friends
5. They can open the link and see your Holi wish

---

## 🛠 Tech Stack

- Next.js
- React
- JavaScript
- Vercel (Deployment)

---

## 🌸 Example

Send a colorful Holi greeting like:

> "Wishing you a Holi filled with colors, happiness, and sweet moments! 🌈"

Generate the link and share it with your loved ones.
